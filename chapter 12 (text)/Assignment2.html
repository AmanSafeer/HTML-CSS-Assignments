Use of Opacity

Opacity property which allows you to speicify the opacity of an element and any of child elements.
When using the opacity property to the background of an element, all of its child elements become transparent as well. This can make the next inside a fully trasparent element hard to read.
Here opacity sets to 0.4{opacity:0.4}

Use of rgba

The rgba will only affect the element on which it is applied(not child element).
when you use alpha, you are only setting opacity for that spiecific property of the element and not it's contents.
Here alpha value sets to 0.4{rgba(155,155,241,0.4)}
